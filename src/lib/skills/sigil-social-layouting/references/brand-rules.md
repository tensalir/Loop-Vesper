# Brand rules (Loop) for social creatives

- **Display/headlines**: Avantt (bold weight). Fallback Space Grotesk.
- **Body/legal**: Space Grotesk. Weights 400 (body), 500–600 (subhead), 700 (emphasis).
- **Colors**: Prefer semantic tokens — primary (purple #6B26B8 light / mint #B3FFD1 dark), foreground (#1A1A1A light / #FAFAFA dark), muted (#737373 light / #999999 dark). Ensure 4.5:1 contrast for text.
- **Spacing**: 4px base; allow breathing room between blocks; align to grid where possible.
- **CTA**: Minimum ~44px height, 120px width; clearly tappable; primary or accent color.
